# Expense Tracker

### To get started
<li>Fork the repo</li>
<li>Clone it to your system</li>
<li>Type the following commands in your terminal</li><br/>

```
npm install
```
```
npm start
```
Open [http://localhost:3300/](http://localhost:3300/) to see the result.
